# ToyVerse Project

## Project Name
**ToyVerse – Online Toy Marketplace**

## Purpose
ToyVerse is a dynamic online platform that allows users to browse, explore, and purchase toys for kids. It is designed for families to easily discover local toy sellers, view detailed toy information, and leave feedback for toys they are interested in buying.

## Live URL
[Live Demo](https://toy-verse-rimi-1234.netlify.app)  

## Key Features
- **User Authentication:** Users can sign up, log in, and log out using email/password or Google login.  
- **Browse Toys:** Users can browse a curated list of toys with details such as name, price, ratings, and description.  
- **Popular Toys Section:** Highlights the most popular toys.  
- **Shop by Age Section:** Recomdation website based on the child’s age group.  
- **Toy Details:** Users can click on a toy to see detailed information.  
- **Customer Reviews:** Users can leave feedback for toys.  
- **Responsive Design:** Fully responsive and mobile-friendly UI.  
- **Password Verification:** Ensures secure login and registration.  
- **Notifications:** Integrated with Toast notifications for actions like login, sign-up, and errors.

## Technologies & NPM Packages Used
- **Frontend:** React 19, React Router v6, TailwindCSS, DaisyUI  
- **Backend / Auth:** Firebase Authentication, Firebase Firestore  
- **Utilities:** react-toastify, custom useTitle hook .lucide-react,react-icons/fa 
- **Other:** Vite (Development & Build Tool)
